    설문조사

- C++ 언어의 기능

1) C++에서는 선언되지 않은 함수를 이용하러면 에러가 발생한다.

2) C++에서는 값을 리턴하도록 선언되어 있는 함수에서 값을 리턴하지 않으면 에러가 발생 한다.

3) C++와 ANSI C는 문자 배열을 초기화할 때 사용하는 문자열 상수의 형을 서로 다르게 해석한다. C++에서는 문자열 상수를 종료시키는 널 문자를 위한 공간을 제공하지 않기 때문이다.

- C++와 C의 몇 가지 차이점

1) C에서 함수에 매개변수가 없을 때 원형(프로토타입)의 매개변수 리스트에 void를 사용해야 했지만 C++에서는 void를 사용하지 않아도 된다.

2) C++ 프로그램에서는 모든 함수가 반드시 원형으로 선언되어야 한다. C에서도 원형을 선언할 수 있지만 기능상으로 선택사항이다. 그러나 멤버 함수의 원형은 클래스 내에서 일반적인 원형으로 선언되므로 별개의 다른 원형을 선언할 필요는 없다.

3) C++에서는 함수가 변환 값을 가지고 있다고 선언하였으면 반드시 값을 반환해야한다. 이것은 함수가 void가 아닌 반환 값을 가지고 있다면 함수 내에 값을 반환하는 return문이 반드시 존재해야 한다는 것을 의미한다. C에서는 void 함수가 아니더라도 값을 반환하지 않으면 의미 없는 값이 반환되므로 실제로 값을 반환하지 않아도 된다.

4) C에서 지역변수는 그것을 사용하는 문장 이전에 블록의 시작에서만 선언되어야 했다. 그러나 C++에서는 어느 곳에서나 지역변수를 선언할 수 있다. 이러한 접근방법은 지역변수가 처음 사용되는 부분 가까이에서 지역변수를 선언할 수 있기 때문에 원하지 않는 부작용을 줄일 수 있게 된다.
